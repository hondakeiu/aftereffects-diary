    Today, for the first time, I moved two properties together, so the speed graph also has two colors! The first color was red and the second was aqua, what will be the third color?

    The best thing about After Effects is that we can always see as many previews as we want without having to render it. When we make animations in gsap, we don't know what the animation will look like until we save the file and play it in the web browser, and can't pause it where we want. So, we have to reload the browser many times, set timescale to 0.1 instead of frame by frame, and stare at the screen to make sure there are no unnatural moments, but we don't have to do that here!
