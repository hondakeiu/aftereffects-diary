  10. - 010

    Now I tried to draw a calm wave. Perhaps a little too calm?

    The effect I used was “Wave Warp” from distortion category. This is exactly the right effect to do this!

    On its own, the boundaries of the waves were a bit jaggy. Applying “Fast Box Blur” effect in “Blur & Sharp” category to blur them, then further adjusting “Alpha” channel with “Curves” effect in “Color Correction” category, clean them up nicely. However, Unless you're viewing this on a very large screen with super high resolution, it doesn't matter. So I didn't do this work in the end this time.
